End-to-end Power BI dashboard for sales, inventory, and supply chain analysis, featuring KPIs, product performance insights, and trend visualization.
# 📊 Power BI Sales, Inventory & Supply Chain Dashboard

# Overview
This project focuses on analyzing **sales performance, inventory management, and supply chain efficiency** using an interactive Power BI dashboard.  
The goal is to provide a clear, data-driven view of business operations and enable quick decision-making through visual insights.

 Tool Used
- Power BI

 Key Features

- Warehouse Utilization Tracking
  Displays warehouse usage percentage to quickly understand capacity and efficiency.

- Inventory Turnover Ratio
  Measures how efficiently inventory is being sold and replaced over time.

- Sales Trend Analysis
  Visualizes units sold over different time periods to identify patterns and growth.

- Backorder Analysis by Category 
  Highlights product categories with higher backorders to identify supply issues.

- Transportation Cost Analysis
  Shows total logistics cost, helping identify areas to optimize expenses.

- KPI Cards for Quick Insights 
  Key metrics displayed for fast and effective decision-making...



# Key Insights

- Certain product categories show **higher backorders**, indicating supply chain inefficiencies.  
- **Inventory turnover ratio** helps identify slow-moving vs fast-moving products.  
- **Transportation costs** contribute significantly to overall expenses and require optimization.  
- Sales trends reveal patterns that can help in forecasting demand.
